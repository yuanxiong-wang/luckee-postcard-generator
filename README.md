# Luckee Seasonal Postcard Generator

A beautiful, interactive web application for generating seasonal greeting postcards for every US and UK holiday.

## Quick Start

### Option 1: Using Python (Easiest)

```bash
python3 -m http.server 8000
```

Then open your browser to: `http://localhost:8000`

### Option 2: Using Node.js

```bash
npx http-server
```

### Option 3: Deploy Online

- **Cloudflare Pages**: Drag and drop the `standalone` folder
- **Netlify**: Drag and drop the `standalone` folder
- **Vercel**: Run `vercel --prod`

See `DEPLOYMENT.md` for detailed instructions.

## Features

✨ **Generate Postcards** - Create unique seasonal greetings for 10 holidays
✏️ **Customize Messages** - Edit greeting text with locked "From our team at, Luckee" signature
📥 **Download** - Export as PNG or PDF for printing and sharing
📱 **Social Sharing** - Share directly to LinkedIn and Facebook
❤️ **Favorites** - Save and manage favorite postcard designs
📅 **Holiday Calendar** - Browse all holidays in a visual calendar
📧 **Email Campaigns** - Compose and preview email campaigns
🌍 **Multi-Region** - Support for US and UK holidays
